# A Hybrid Model Including Bi-RNN,Attention Layer and Lexicon-Based Feature Extraction For Detecting Frustration On Online Product Reviews 
Bachelor Thesis Project

## Overview
This research proposes a hybrid deep learning model for detecting frustration in online product reviews using:
- Bi-RNN
- Attention Layer
- Lexicon-based feature extraction

## Technologies
- Python
- TensorFlow
- NLP
- Deep Learning
